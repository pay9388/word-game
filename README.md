# 조씨네 영어 학습방

초등학생 자녀를 위한 영단어 학습 웹앱입니다.

---

## 주요 기능

### 학생 화면
- 이름 선택 로그인 (조연우 / 조연수)
- 단원별 단어 학습
- **플래시카드** - 단어 카드를 눌러 뜻 확인, 🔊 발음 듣기
- **스펠링 퀴즈** - 뜻을 보고 영어 단어 입력
- **틀린 단어 학습** - 틀린 단어만 모아 다시 플래시카드 → 스펠링 반복
- 모두 맞히면 완료 화면 + 딩동댕 효과음
- 정답/오답 효과음 제공
- 단원 완료 기록 저장

### 관리자 화면 (아빠)
- 로그인 화면에서 **아빠** 버튼 → 비밀번호 입력
- 아이별 단원 추가 / 삭제
- 단어 하나씩 추가 또는 **CSV 파일 일괄 업로드**
- 샘플 CSV 파일 다운로드 제공

|------|-----|

## 기술 스택

- **호스팅**: GitHub Pages (무료)
- **데이터베이스**: Firebase Firestore (무료)
- **발음**: Web Speech API TTS (브라우저 내장, 무료)

---

## 접속 방법

- **학생**: 메인 주소로 접속 후 이름 선택
- **관리자**: 메인 주소 접속 → 아빠 버튼 → 비밀번호 입력
